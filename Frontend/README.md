⚙️ Installation Guide
1️⃣ Clone the Repository
git clone https://github.com/mukeshkanna11/ReadyTech-CRM-Dashboard
cd crm-project

2️⃣ Backend Setup
cd backend
npm install


Create .env file:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


Run backend:

npm run dev

3️⃣ Frontend Setup
cd frontend
npm install
npm run dev
